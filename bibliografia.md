## Documentação Oficial e Especificações

- **MDN Web Docs.** Recursos para desenvolvedores, por desenvolvedores. Mozilla Foundation. Disponível em: [developer.mozilla.org/pt-BR](https://developer.mozilla.org/pt-BR/). Acesso em: 2025.

- **W3C - World Wide Web Consortium.** Web Standards. Disponível em: [w3.org](https://www.w3.org/). Acesso em: 2025.

- **WHATWG.** HTML Living Standard. Disponível em: [html.spec.whatwg.org](https://html.spec.whatwg.org/). Acesso em: 2025.

- **W3C.** CSS Specifications. Disponível em: [w3.org/Style/CSS](https://www.w3.org/Style/CSS/). Acesso em: 2025.

- **ECMA International.** ECMAScript Language Specification (ECMA-262). Disponível em: [ecma-international.org](https://www.ecma-international.org/publications-and-standards/standards/ecma-262/). Acesso em: 2025.

- **Git.** Documentação Oficial do Git. Disponível em: [git-scm.com/doc](https://git-scm.com/doc). Acesso em: 2025.

- **GitHub Docs.** Documentação do GitHub. Disponível em: [docs.github.com](https://docs.github.com/pt). Acesso em: 2025.
