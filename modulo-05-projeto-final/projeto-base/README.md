# Meu Portfólio - Projeto Final

> Projeto final do curso **Fundamentos de Desenvolvimento Web**.

## Sobre o Projeto

Este é um site de portfólio pessoal desenvolvido como projeto final do curso. O site utiliza HTML5 semântico, CSS3 com Flexbox e Grid, e JavaScript para interatividade.

## Páginas

- **Guia** (`index.html`) - Página de orientação do projeto base
- **Exemplo Visual** (`exemplo-visualizacao.html`) - Apresentação, projetos e habilidades (antigo `index.html`)
- **Sobre** (`sobre.html`) - Informações pessoais e formação
- **Contato** (`contato.html`) - Formulário de contato com validação JavaScript

## Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, variáveis CSS, media queries)
- JavaScript (manipulação do DOM, eventos, validação de formulários)
- Git & GitHub

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/seu-repositorio.git
   ```

2. Abra o arquivo `index.html` no navegador.

## Personalização

1. Substitua "Seu Nome" pelo seu nome em todos os arquivos HTML
2. Adicione suas próprias imagens na pasta `img/`
3. Personalize as cores no arquivo `css/style.css` (variáveis CSS no `:root`)
4. Atualize os links de redes sociais no rodapé e na página de contato
5. Adicione seus projetos reais na seção de projetos da Home
