# Exemplo de Issue Inicial (Dia 1)

Use este modelo para abrir uma issue de exemplo no repositório.

## Título sugerido

`Check-in Dia 1 - Turma Tarde - Seu Nome`

ou

`Check-in Dia 1 - Turma Noite - Seu Nome`

## Labels sugeridas

- `tipo:atividade`
- `turma:tarde` ou `turma:noite`
- `status:triagem`

## Corpo da Issue (copiar e preencher)

```md
## Check-in Dia 1

**Nome:** Seu Nome
**Turma:** Tarde/Noite
**Módulo atual:** Módulo 1
**Status:** Presente

### O que já fiz hoje
- [x] Li o material do Módulo 1
- [x] Fiz minha apresentação na Discussion fixada
- [x] Configurei ambiente local (editor + navegador)

### Dificuldade inicial (opcional)
Descreva aqui se teve alguma dificuldade no primeiro dia.

### Próximo passo
Continuar para os exercícios práticos do módulo.
```

## Objetivo deste exemplo

- Mostrar o padrão de abertura de issue para a turma.
- Servir de referência para os próximos check-ins.
