CSS Box Model

Todo elemento HTML é uma "caixa" composta por 4 camadas: Content, Padding, Border e Margin.

Visualização do Box Model

MARGIN (espaçamento externo - laranja)
BORDER (borda - vermelha)
PADDING (espaçamento interno - verde)
CONTENT (conteúdo - azul)

content-box vs border-box

content-box (padrão): width/height = apenas o conteúdo. Padding e border são adicionados.

width: 300px + padding: 40px + border: 10px = 350px total

border-box (recomendado): width/height = conteúdo + padding + border.

width: 300px total (padding e border inclusos)

Dica: Sempre use box-sizing: border-box no seletor universal (*) para facilitar cálculos!
