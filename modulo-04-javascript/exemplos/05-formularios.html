📝 Formulários com JavaScript

Módulo 4 - JavaScript | Exemplo 05

1. Lendo Valores de Campos

Use a propriedade .value para obter o que o usuário digitou:

// Ler valor de um input const nome = document.querySelector('#nome').value; // Ler checkbox (checked é boolean) const aceito = document.querySelector('#aceito').checked; // Ler select (valor da opção selecionada) const cidade = document.querySelector('#cidade').value;

2. Interceptando o Envio: preventDefault

Por padrão, um formulário recarrega a página ao ser enviado. Use preventDefault para evitar isso e processar com JavaScript.

form.addEventListener('submit', function(e) { e.preventDefault(); // Impede o reload da página! // Agora processar os dados com JavaScript const nome = document.querySelector('#nome').value; console.log("Formulário interceptado!", nome); });
💡 Sem preventDefault: A página recarregaria e os dados seriam perdidos. Com ele, processamos tudo no JavaScript!
