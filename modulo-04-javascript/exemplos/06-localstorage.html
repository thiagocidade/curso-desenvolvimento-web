Módulo 4 - JavaScript | Exemplo 06

1. O que é localStorage?

O localStorage permite salvar dados no navegador que persistem mesmo depois de fechar a página. É como um "mini banco de dados" local.

// SALVAR um dado localStorage.setItem("nome", "Maria"); // LER um dado const nome = localStorage.getItem("nome"); // "Maria" // REMOVER um dado localStorage.removeItem("nome"); // LIMPAR tudo localStorage.clear();

Salve um valor e veja que ele persiste ao recarregar a página:


⚠️ Importante: O localStorage só armazena strings! Para salvar objetos ou arrays, use JSON.stringify() e JSON.parse().

2. Salvando Objetos e Arrays (JSON)

// Salvar um objeto const usuario = { nome: "Ana", idade: 25 }; localStorage.setItem("usuario", JSON.stringify(usuario)); // Recuperar o objeto const dados = JSON.parse(localStorage.getItem("usuario")); console.log(dados.nome); // "Ana" // Salvar um array const frutas = ["Maçã", "Banana", "Laranja"]; localStorage.setItem("frutas", JSON.stringify(frutas));
