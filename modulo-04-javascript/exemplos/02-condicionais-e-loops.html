2. Operador Ternário

Uma forma compacta de escrever um if/else simples em uma única linha.

// Sintaxe: condição ? valorSeVerdadeiro : valorSeFalso let idade = 18; let status = idade >= 18 ? "Maior de idade" : "Menor de idade"; // É o mesmo que: // if (idade >= 18) { status = "Maior de idade"; } // else { status = "Menor de idade"; }

3. switch

Ideal quando você precisa comparar uma variável com vários valores possíveis.

switch (diaDaSemana) { case 1: texto = "Segunda"; break; case 2: texto = "Terça"; break; case 3: texto = "Quarta"; break; // ... etc default: texto = "Dia inválido"; }

4. Loop for

O for repete um bloco de código um número determinado de vezes.

// Estrutura: for (início; condição; incremento) for (let i = 1; i <= 5; i++) { console.log(`Número: ${i}`); } // Imprime: 1, 2, 3, 4, 5

5. Loop while

O while repete enquanto a condição for verdadeira. Use quando não sabe exatamente quantas vezes repetir.

// Contagem regressiva let contador = 5; while (contador > 0) { console.log(contador); contador--; // diminui 1 a cada volta } console.log("🚀 Lançar!");

⚠️ Cuidado: Um loop while sem condição de parada causa um loop infinito que trava o navegador! Sempre garanta que a condição vai se tornar falsa em algum momento.

6. for...of (Percorrer Arrays)

O for...of é a forma mais simples de percorrer todos os itens de um array.

const frutas = ["Maçã", "Banana", "Laranja", "Manga"]; for (const fruta of frutas) { console.log(`Fruta: ${fruta}`); } // Imprime cada fruta, uma por vez
