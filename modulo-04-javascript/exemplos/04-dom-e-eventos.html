1. Selecionando Elementos: querySelector

Para modificar a página com JavaScript, primeiro precisamos selecionar os elementos HTML.

// Seleciona o PRIMEIRO elemento que corresponde ao seletor CSS const titulo = document.querySelector('h1'); const botao = document.querySelector('.btn-principal'); const menu = document.querySelector('#menu-nav'); // Seleciona TODOS os elementos const itens = document.querySelectorAll('.item'); const parags = document.querySelectorAll('p');

Este é um parágrafo de demonstração.

2. Modificando Conteúdo: textContent e innerHTML

// textContent - altera só o texto (mais seguro) elemento.textContent = "Novo texto"; // innerHTML - altera o HTML interno (cuidado com XSS!) elemento.innerHTML = "<strong>Texto em negrito</strong>";

Clique nos botões para me modificar!


3. Manipulando Classes: classList

Use classList para adicionar, remover ou alternar classes CSS dinamicamente.

elemento.classList.add('destaque'); // adiciona elemento.classList.remove('destaque'); // remove elemento.classList.toggle('destaque'); // alterna (add/remove) elemento.classList.contains('destaque'); // verifica (true/false)

Observe minhas classes mudarem!
