1. Declarando Variáveis: let e const

let cria uma variável que pode ter seu valor alterado. const cria uma constante (valor fixo).

// Variável - pode mudar let nome = "Maria"; nome = "João"; // ✅ OK - let permite reatribuir // Constante - NÃO pode mudar const PI = 3.14159; PI = 3; // ❌ ERRO! const não permite reatribuir
⚠️ E o var? O var é a forma antiga de declarar variáveis. Evite usar — ele tem comportamentos confusos com escopo. Use sempre let ou const.

2. Os Tipos de Dados do JavaScript

JavaScript tem tipos primitivos (simples) e tipos compostos (complexos).

// Tipos primitivos let texto = "Olá mundo"; // string (texto) let idade = 25; // number (número) let ativo = true; // boolean (verdadeiro/falso) let vazio = null; // null (vazio intencional) let indefinido = undefined; // undefined (não definido)

3. Descobrindo o Tipo: typeof

O operador typeof retorna o tipo de um valor como texto.

typeof "Olá" // "string" typeof 42 // "number" typeof true // "boolean" typeof undefined // "undefined" typeof null // "object" ← Bug histórico do JS! typeof [1, 2, 3] // "object" ← Arrays são objetos
💡 Curiosidade: typeof null retorna "object" — isso é um bug que existe desde 1995! Nunca foi corrigido para não quebrar sites antigos.

4. Trabalhando com Strings (Textos)

Strings são textos. Podem usar aspas simples, duplas ou crases (template literals).

let aspasSimples = 'Olá mundo'; let aspasDuplas = "Olá mundo"; let templateLiteral = `Olá mundo`; // crases - aceita expressões! let nome = "Maria"; let saudacao = `Olá, ${nome}!`; // "Olá, Maria!"

6. Booleans (Verdadeiro ou Falso)

Boolean é o tipo mais simples: só pode ser true ou false. É fundamental para condicionais.

let maiorDeIdade = true; let chovendo = false; // Comparações retornam booleans 10 > 5 // true 3 === "3" // false (tipos diferentes) 3 == "3" // true (compara só o valor - EVITE!)
⚠️ Importante: Use sempre === (igualdade estrita) em vez de == (igualdade solta). O === compara valor E tipo, evitando surpresas.

7. Conversão de Tipos

Às vezes precisamos converter um tipo em outro. JavaScript oferece funções para isso.

// Para número Number("42") // 42 Number("abc") // NaN (Not a Number) Number(true) // 1 // Para string String(42) // "42" String(true) // "true" // Para boolean Boolean(1) // true Boolean(0) // false Boolean("") // false Boolean("texto") // true
